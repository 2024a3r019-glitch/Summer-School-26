# Arduino LED Blink Project 💡

## Overview
This project demonstrates a basic LED blinking program using Arduino.  
The built-in LED on the Arduino board is turned ON for 1 second and then turned OFF for 1 second repeatedly.

This is a beginner-level project to understand:
- Arduino programming
- Digital output control
- Delay functions
- Basic hardware interaction

## Components Required
- Arduino Board (Arduino UNO / compatible)
- USB Cable
- Arduino IDE
